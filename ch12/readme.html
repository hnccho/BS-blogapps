RSS And Atom In Action - Chapter 12 Technorati API example

Technorati API example that generates an OPML file containing all of the Technorati blog cosmos, link cosmos or outbound links found for the blog at specified URL.

Build the example

First, use Ant to build the example to the dist/tapi2opml directory.

For example on UNIX: 
   $ cd $BLOGAPPS/java/ch12
   $ ant
Or on Windows: 
   > cd $BLOGAPPS\examples\java\ch12
   > ant

Run the example

Before you can run the example you'll need to get a Technorati API key. Go to http://technorati.com/developers and sign up for the developer program to get a free 500 query per day key. Put that key into the dist/tapi2opml directory so that it will be on the classpath for tapi2opml.

To run the example and change directories into the dist/tapi2opml directory. Run the example using either tapi2opml.bat (for Windows) or tapi2opml.sh (for UNIX).

The program takes three arguments: the type of query (blogcosmos, linkcosmos or outbound), the URL of the target blog and a filename where results are to be stored.

tapi2opml.sh [blogcosmos|linkcosmos|outbound] [search] [filename]

For example on UNIX:

   $ cd $BLOGAPPS/examples/java/ch12/dist/tapi2opml
   $ ./tapi2opml.sh blogcosmos http://rollerweblogger.org/page/roller daveslinks.opml

Or on Windows:

   > cd $BLOGAPPS\examples\java\ch12\dist\tapi2opml
   > tapi2opml blogcosmos http://rollerweblogger.org/page/roller daveslinks.opml
