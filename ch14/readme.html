RSS And Atom In Action - Chapter 14 mail blogger example

Mail blogger monitors a POP3 mailbox, looking for emails that start with a codeword and when one is found it posts it to a blog.

Build the example

First, use Ant to build the example to the dist/mailblogger directory.

For example on UNIX: 
   $ cd $BLOGAPPS/java/ch14
   $ ant
Or on Windows: 
   > cd %BLOGAPPS%\java\ch14
   > ant

Run the example

To run the example and change directories into the dist/mailblogger directory. Run the example using either MailBlogger.bat (for Windows) or MailBlogger.sh (for UNIX).

For example on UNIX: 
   $ cd $BLOGAPPS/java/ch14/dist/mailblogger
   $ vi MailBlogger.config.xml
   $ ./MailBlogger.sh MailBlogger.config.xml
Or on Windows: 
   > cd %BLOGAPPS%\java\ch014\dist\mailblogger
   > notepad MailBlogger.config.xml
   > MailBlogger MailBlogger.config.xml

The vi and notepad lines are there because you will want to edit the MailBlogger.config.xml file to set your blog codeword, POP3 connection parameters and blog connection information.
