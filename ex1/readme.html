RSS And Atom In Action - Extra 1 FeedPoster example

Command-line program reads a list of newsfeeds, parses each newsfeed, collects all entries published in the last X days, and posts a summary of those posts to a destination blog.

Build the example

Use Ant to build the examples to the dist/feedposter directory.

For example on UNIX: 
   $ cd $BLOGAPPS/java/ex01
   $ ant
Or on Windows: 
   > cd %BLOGAPPS%\java\ex01
   > ant

Run the example

To run the example, change directories into the dist/feedposter directory and edit the FeedPoster.config.xml file to set your newsfeed URLs as well as your blog username, password, blog server URL and blog ID. Then you can run the example using either FeedPoster.bat (for Windows) or FeedPoster.sh (for UNIX).

For example on UNIX: 
   $ cd $BLOGAPPS/java/ex01/dist/feedposter
   $ vi FeedPoster.config.xml
   $ ./FeedPoster.sh
Or on Windows: 
   > cd %BLOGAPPS%\java\ex01\dist\feedposter
   > notepad FeedPoster.config.xml
   > FeedPoster

The vi and notepad lines are there because you'll want to edit FeedPoster.config.xml as we described above. The default settings are for the Blogapps server.
