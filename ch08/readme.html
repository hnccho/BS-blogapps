RSS And Atom In Action - Chapter 8 feed generation examples

Simple FileDepot file download web page with an RSS newsfeed

Build the example

Use Ant to build the examples to the dist/filedepot directory. The build script will compile the code and build the dist/filedepot.war, which is ready for deployment.

For example on UNIX: 
   $ cd $BLOGAPPS/java/ch08
   $ ant
Or on Windows: 
   > cd %BLOGAPPS%\java\ch08
   > ant

Deploy and run example

The Chapter 6 examples are part of a Java web application called "filedepot". To run filedepot, you must first deploy it to a Java web application server. You can use the Blogapps demo server, which just a customized version of Tomcat. To install to Tomcat or Blogapps server, just copy the filedepot WAR to the webapps directory.

For example on UNIX: 
   $ cd $BLOGAPPS/java/ch08/dist
   $ cp -r filedepot.war [TOMCAT-DIR]/webapps
Or on Windows: 
   $ cd %BLOGAPPS%\java\ch08\dist
   $ copy filedepot.war [TOMCAT-DIR]\webapps

If you're using Blogapps server, them use it's directory as the TOMCAT-DIR.

Next, start (or restart) Tomcat and then open http://localhost:8080/filedepot in your browser to see the filedepot examples.
