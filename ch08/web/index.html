Six different versions of the filedepot newsfeed.

Atom 1.0 DepotAtomServlet.java /depot-atom Generated by Velocity (see atom.vm in WEB-INF/classes) and styled with CSS.
RSS 1.0 DepotNewsfeedServlet.java /depot-newsfeed?format=rss_1.0 Generated by Rome
RSS 2.0 DepotNewsfeedServlet.java /depot-newsfeed?format=rss_2.0 Generated by Rome
RSS 2.0 DepotNewsfeedServletCached.java /depot-newsfeed-cached?format=rss_2.0 Generated by Rome, cached by LRUCache
RSS 2.0 depot-rss.jsp /depot-rss.jsp Generated by JSP, styled with CSS
RSS 2.0 depot-rss-cached.jsp /depot-rss-cached.jsp Generated by JSP, cached by OSCache, styled with CSS
