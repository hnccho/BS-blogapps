RSS And Atom In Action - Chapter 13 dialy cross poster example

Simple newsfeed parsing and blog entry posting example that reads a list of newsfeeds and posts the last 24 hous of entries from those newsfeeds to a blog.

Build the example

First, use Ant to build the example to the dist/crossposter directory.

For example on UNIX: 
   $ cd $BLOGAPPS/java/ch13
   $ ant
Or on Windows: 
   > cd %BLOGAPPS%\java\ch13
   > ant

Run the example

To run the example and change directories into the dist/crossposter directory. Run the example using either CrossPoster.bat (for Windows) or CrossPoster.sh (for UNIX).

For example on UNIX: 
   $ cd $BLOGAPPS/java/ch13/dist/crossposter
   $ vi CrossPoster.config.xml
   $ ./CrossPoster.sh CrossPoster.config.xml
Or on Windows: 
   > cd %BLOGAPPS%\java\ch13\dist\crossposter
   > notepad CrossPoster.config.xml
   $ CrossPoster CrossPoster.config.xml

The vi and notepad lines are there because you will want to edit the CrossPoster.config.xml to specify the list of source newsfeeds to be read and the destination blog for the blog posts.
