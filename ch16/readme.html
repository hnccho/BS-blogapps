RSS And Atom In Action - Chapter 16 Ant blogging tasks example

Ant tasks for posting entries and uploading files to a blog:

   post-blog-resource 
   post-blog-entry

Build the example

First, use Ant to build the example to the dist/blogtasks directory.

For example on UNIX: 
   $ cd $BLOGAPPS/java/ch16 
   $ ant
Or on Windows: 
   > cd %BLOGAPPS%\java\ch16
   > ant

Run the example

Refer to dist/blogtasks/example.xml for an example of how to set the classpath for, define and use the blogging tasks in an Ant script. The example shows how to upload an image and then create a blog entry that includes that image.
