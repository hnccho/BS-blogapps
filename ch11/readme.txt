
RSS And Atom In Action Chapter 9 Example
========================================

Use Ant to build the example to the dist/planet-roller directory.

For example on UNIX:
   $ cd <INSTALL-DIR>/java
   $ ant

Or on Windows:
   > cd <INSTALL-DIR>\java
   > ant

To run the example and change directories into the dist/planet-roller 
directory. Run the example using either planet-tool.bat (for Windows) 
or planet-tool.sh (for UNIX). 

For example on UNIX:
   $ cd <INSTALL-DIR>/java
   $ ant
   $ cd ch09/dist/planet-roller
   $ chmod +x ./planet-tool.sh
   $ ./planet-tool.sh planet-config.xml

Or on Windows:
   > cd <INSTALL-DIR>\java
   > ant
   > cd ch09\dist\planet-roller
   > planet-tool planet-config.xml
