RSS And Atom In Action - Chapter 11 planet aggregator example

Generates a planet-style aggregation of weblogs with an HTML, RSS and OPML page. Configured via a simple XML file. You can customize the output by editing the Velocity templates that define the outputs.

Build the example

First, use Ant to build the example to the dist/planet-tool directory.

For example on UNIX: 
   $ cd $BLOGAPPS/java/ch11
   $ ant
Or on Windows: 
   > cd %BLOGAPPS%\java\ch11
   > ant

Run the example

To run the example and change directories into the dist/planet-tool directory. Run the example using either planet-tool.bat (for Windows) or planet-tool.sh (for UNIX).

For example on UNIX: 
   $ cd $BLOGAPPS/java/ch11/dist/planet-tool
   $ vi planet-config.xml
   $ ./planet-tool.sh planet-config.xml
Or on Windows: 
   > cd %BLOGAPPS%\java\ch11\dist\planet-tool
   > notepad planet-config.xml
   > planet-tool planet-config.xml

The vi and notepad lines are there because you will want to edit the planet-config.xml file to add the newsfeed URLs you want to include in the planet. You might also want to edit the Velocity templates that define the HTML, RSS and OPML that is generated in the templates directory.
