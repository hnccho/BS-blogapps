RSS And Atom In Action - Chapter 15 blog mailer example

Blog mailer reads a list of newsfeeds, collects all of the entries made in the last 72 hours, puts them in a digest email and sends them to an email address.

Build the example

First, use Ant to build the example to the dist/blogmailer directory.

For example on UNIX: 
   $ cd $BLOGAPPS/java/ch15
   $ ant
Or on Windows: 
   > cd %BLOGAPPS%\java\ch15
   > ant

Run the example

To run the example and change directories into the dist/blogmailer directory. Run the example using either BlogMailer.bat (for Windows) or BlogMailer.sh (for UNIX).

For example on UNIX: 
   $ cd $BLOGAPPS/java/ch15/dist/blogmailer
   $ vi BlogMailer.config.xml
   $ ./BlogMailer.sh BlogMailer.config.xml
Or on Windows: 
   > cd %BLOGAPPS%\java\ch15\dist\blogmailer
   > notepad BlogMailer.config.xml
   > BlogMailer BlogMailer.config.xml

The vi and notepad lines are there because you will want to edit the BlogMailer.config.xml file add the list of newsfeeds and the destination blog connection parameters.
