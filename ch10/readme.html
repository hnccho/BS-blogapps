RSS And Atom In Action - Chapter 10 Atom examples

In Chapter 10 we developed a set of blog client library interfaces and both an Atom protocol and MetaWeblog API implementations of those interfaces. You can't run a library, so we've included a GUI based blog client that uses the library.

The example code is organized like so:

    $BLOGAPPS/java/ch10/src/com/manning/blogapps/blogclient
    Generic blog client interfaces
    
    $BLOGAPPS/java/ch10/src/com/manning/blogapps/atomblogclient
    Atom protocol implementation of the blog client interfaces
    
    $BLOGAPPS/java/ch10/src/com/manning/blogapps/metaweblogblogclient
    MetaWeblog API implementation of the blog client interfaces
    
    $BLOGAPPS/java/ch10/src/com/manning/blogapps/blogclientui
    Simple blog client GUI, written in Swing

Build the example

First, use Ant to build the example to the dist/blogclient directory.

For example on UNIX: 
   $ cd $BLOGAPPS/java/ch10
   $ ant
Or on Windows: 
   > cd %BLOGAPPS%\java\ch10
   > ant

Run the example

To run the example and change directories into the dist/blogclient directory. Run the example using either blogclient.bat (for Windows) or blogclient.sh (for UNIX).

For example on UNIX: 
   $ cd $BLOGAPPS/java/ch10/dist/blogclient
   $ ./blogclient.sh
Or on Windows: 
   > cd %BLOGAPPS%\java\ch10\dist\blogclient
   > blogclient
