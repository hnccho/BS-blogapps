RSS And Atom In Action - Extra 2 Grabber example

Command-line program that downloads blog entries from a blog, saves each one as an HTML file and creates an index that links to each one.

Build the example

Use Ant to build the examples to the dist/grabber directory.

For example on UNIX: 
   $ cd $BLOGAPPS/java/ex02
   $ ant
Or on Windows: 
   > cd %BLOGAPPS%\java\ex02
   > ant

Run the example

To run the example, change directories into the dist/grabber directory and edit the config.properties file to set your blog username, password and blog server URL, blog ID and the path to the directory into which you'd like to download files. Then you can run the example using either Grabber.bat (for Windows) or Grabber.sh (for UNIX).

For example on UNIX: 
   $ cd $BLOGAPPS/java/ex02/dist/grabber
   $ vi config.properties
   $ ./Grabber.sh
Or on Windows: 
   > cd %BLOGAPPS%\java\ex02\dist\grabber
   > notepad config.properties
   > Grabber

The vi and notepad lines are there because you might want to edit config.properties to set the URL, username and password for your blogs MetaWeblog API interface. The default settings are for the Blogapps demo server.
