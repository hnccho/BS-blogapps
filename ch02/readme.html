RSS And Atom In Action - Chapter 2 Blogposter example - test

Simple MetaWeblog API example that posts to a blog from the command line.

Build the example

First, use Ant to build the example to the dist/blogposter directory.

For example on UNIX: 
   $ cd $BLOGAPPS\java\ch02
   $ ant
Or on Windows: 
   > cd %BLOGAPPS%\java\ch02
   > ant

Run the example

To run the example, change directories into the dist/blogposter directory and edit the config.properties file to set your blog username, password and blog server URL. Then you can run the example using either blogpsoster.bat (for Windows) or blogposter.sh (for UNIX).

For example on UNIX: 
   $ cd $BLOGAPPS/java/ch02/dist/blogposter
   $ vi config.properties
   $ ./blogposter.sh "This is blog title" "This is a description"
Or on Windows: 
   > cd $BLOGAPPS\java\ch02\dist\blogposter
   > notepad config.properties
   > blogposter "This  is a blog title" "This is a description"

The vi and notepad lines are there because you might want to edit config.properties to set the URL, username and password for your blogs MetaWeblog API interface. The default settings are for the Blogapps demo server.
